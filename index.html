Passo a passo — Link Fixo com ngrok (gratuito)
1
Criar conta gratuita e baixar o ngrok.exe
Crie a conta (pode usar Google), baixe o ngrok.exe e coloque o arquivo dentro da mesma pasta do sistema:
C:\Sistemas\mundo-verde-v2-completo\ngrok.exe
Criar conta → Baixar ngrok.exe →
2
Pegar o Authtoken e criar domínio fixo
• Getting Started → Your Authtoken → copie o token
• Cloud Edge → Domains → New Domain → copie o domínio (ex: minha-loja.ngrok-free.app)
3
Gerar o arquivo de inicialização
Cole os dados abaixo e clique em gerar — será baixado o arquivo INICIAR_TUDO.bat já configurado com seu token e domínio. Use ele no lugar do INICIAR.bat.
4
Usar
Salve o INICIAR_TUDO.bat gerado na pasta do sistema.
Da próxima vez, clique só nele — ele abre o servidor e o link externo juntos automaticamente!

Duas janelas vão abrir — mantenha as duas abertas enquanto o sistema estiver em uso.
⚡ Opção rápida — link temporário (sem cadastro)
Funciona na hora, sem criar conta. O link muda quando reiniciar o bat.
